# نحوه‌ی Run کردن کد

پروژه به زبان python نوشته شده.
برای اجرای کد سرور کافیست دستور زیر را در کنسول بزنید:

```bash
make run-server
```

برای اجرای کد کلاینت(به تعداد دلخواه) کافیست دستور زیر را در کنسول‌های مجزا بزنید:

```bash
make run-client
```
